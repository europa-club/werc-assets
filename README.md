# WERC (Euro Finance) — Official Assets

This repository contains the official brand assets of Euro Finance (WERC).

## Logo

Primary logo files:

- logo.png (standard)
- logo@2x.png (high resolution)

## Usage

These assets are the only official logos approved for:

- wallets
- exchanges
- aggregators (CoinMarketCap, Trust Wallet)
- third-party integrations

## Important Notice

- WERC is an independent decentralized asset
- NOT affiliated with Circle
- NOT EURC
- NOT a stablecoin

Any unofficial logos or tokens claiming association with WERC should not be trusted.
